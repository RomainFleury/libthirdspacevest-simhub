## Generic Screen Health Watcher

This folder contains planning + implementation tracking docs for the “Generic Screen Health Watcher” feature (screen-based `hit recorded` / health tracking integration).

### Structure
- `conception/`: background + initial spec + phased plan
- `phase-a/`: MVP (redness ROIs → `hit recorded`) plan/checklists/progress/issues
- `phase-b/`: directional expansions
- `phase-c/`: health bar tracking
- `phase-d/`: health number OCR
- `phase-e/`: capture source expansion (optional)
