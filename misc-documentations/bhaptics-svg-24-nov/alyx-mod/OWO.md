# OWo iontegration example of alyx

MOD – This game requires Mods to connect with the OWO System.

First of all, download and install from the official website: https://github.com/floh-bhaptics/HalLifeAlyx_OWO/releases/tag/v2.0.4 and NexusMod: https://www.nexusmods.com/halflifealyx/mods/15?tab=files.

Installation

Manual:

Extract the contents of the Scripts rar file inside your “Steam\steamapps\common\Half-Life Alyx” directory so that you’ll have a vscripts folder like this: “Steam\steamapps\common\Half-Life Alyx\game\hlvr\scripts\vscripts”.

Extract the contents of the Application rar file anywhere you want.

Run OWOAlyx.exe, set your Half-Life Alyx folder correctly and it’s ready to be used.

Installer:

Execute the application inside of OWOAlyx compress file.

Select your “Half-Life Alyx” installation folder, once selected click on install mod.
