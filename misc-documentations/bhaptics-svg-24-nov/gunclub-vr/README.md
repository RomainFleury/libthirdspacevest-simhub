About this mod
Adds haptic effects for "Gun Club VR" in the bhaptics Tactsuit models, Tactal, and Tactosy for arms and hands. Effects include gun and shotgun recoil and getting shot. This is far from my best mod, the game was horrible to hook into. But I thought it's better than nothing, so I uploaded it anyways. Please do not buy the game because of it!

Installation

Download and install MelonLoader (and the requirements for it if needed) into the game's directory.
Run the game once so the mod can make its folder structure. You can quit right out of it.
Download and extract the zip file of this mod into the "Mods" subdirectory of your game directory.
You are done, just start the bhaptics Player and then the game how you normally would.

If you have any issues or suggestions, the best way to contact me is to ping me on the bhaptics Discord: https://discord.gg/tHB8StRX
Description

This mod communicates with the bhaptics gear when certain events are detected in "Gun Club VR" to give you haptic feedback. These are some of the currently implemented haptic effects:

Recoil on guns and shotguns
Barrel explosions
Getting shot\
